# 🧮 Simple Calculator

A friendly command-line calculator that adds, subtracts, multiplies, and divides. Type 'q' anytime to quit. No more crying over wrong numbers! 😎

## ✨ Features

- Four basic operations: addition (+), subtraction (-), multiplication (*), division (/)
- Input validation for non-numeric entries
- Division by zero protection
- Modular code structure (separate operation modules)
- Quit option at any stage (type 'q')
- Clean and readable error messages

## 🚀 How to Run

1. Make sure you have Python installed (version 3.6+)

2. Run the program:
```bash
python calculator.py
```
